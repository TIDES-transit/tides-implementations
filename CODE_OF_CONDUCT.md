# Code of Conduct

This repository follows the [TIDES Code of Conduct](https://github.com/TIDES-transit/TIDES/blob/main/docs/governance/policies/code-of-conduct.md), which applies to all community spaces in the TIDES project.

## Summary

Participants in this repository agree to help keep the TIDES community welcoming, respectful, and collaborative. Contributors, maintainers, and reviewers are expected to interact in good faith, give and receive feedback constructively, and focus on what is best for the community as a whole.

Behavior that is harassing, discriminatory, or otherwise inappropriate in a professional setting is not welcome. This includes personal attacks, publishing others' private information without consent, and any conduct that makes participation in the project feel unsafe.

## Scope

This Code of Conduct applies in all spaces associated with this repository, including issues, pull requests, discussions, and any community venues where participants are representing the TIDES project.

## Reporting

If you experience or witness conduct that you believe violates the TIDES Code of Conduct, please contact the TIDES community leaders at [tidestransit@gmail.com](mailto:tidestransit@gmail.com).

Reports are reviewed promptly and handled with respect for the privacy of the reporter.

## Enforcement

The full enforcement guidelines, including the specific responses available to community leaders, are documented in the [TIDES Code of Conduct](https://github.com/TIDES-transit/TIDES/blob/main/docs/governance/policies/code-of-conduct.md). Contributors are expected to read that document.
